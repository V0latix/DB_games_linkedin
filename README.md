# DB_games_linkedin

Base finale des puzzles Zip extraits depuis des vidéos YouTube LinkedIn Games.

## Contenu

- `data/zip_unique.json`: dataset au format exact utilisé par le site (`game`, `version`, `puzzles`).

## Notes

- Source: extraction vidéo YouTube (pas de scraping LinkedIn).
- Le repo est volontairement minimal et autosuffisant pour consommation de la base finale.
